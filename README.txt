OBS TITLES PRO - GITHUB PAGES

Archivos incluidos:
- index.html
- manifest.json
- service-worker.js
- icons/icon-192.png
- icons/icon-512.png

PASOS PARA SUBIR A GITHUB:
1. Crear un repositorio nuevo.
2. Subir todos estos archivos a la raiz del repositorio.
3. Ir a Settings > Pages.
4. En Source elegir Deploy from a branch.
5. Seleccionar branch main y carpeta /(root).
6. Guardar.
7. Esperar que GitHub genere la URL publica.

COMO USAR EN OBS:
- URL del panel:
  https://TU-USUARIO.github.io/TU-REPO/
- URL del overlay:
  https://TU-USUARIO.github.io/TU-REPO/?view=overlay

EN OBS:
1. Agregar Browser Source.
2. Pegar la URL con ?view=overlay
3. Width: 1920
4. Height: 1080
5. Activar transparencia si corresponde.

INSTALACION COMO APP:
- En Chrome/Edge suele aparecer el boton instalar.
- En Android se puede usar "Agregar a pantalla de inicio".
- En PC se instala como app independiente.
