PODCAST OVERLAY STUDIO - GITHUB PAGES

1) Subi todo este contenido a un repositorio nuevo de GitHub.
2) En GitHub: Settings > Pages > Deploy from a branch.
3) Elegi branch main y carpeta /root.
4) Espera la URL pública.

USO:
- Panel de control: https://TU-USUARIO.github.io/TU-REPO/
- URL para OBS: https://TU-USUARIO.github.io/TU-REPO/?view=overlay

EN OBS:
- Crear fuente Browser Source
- Pegar la URL con ?view=overlay
- Ancho: 1920
- Alto: 1080

FUNCIONES:
- Web app instalable
- Estilo podcast moderno
- Carga de logo
- Mostrar/ocultar logo, títulos, lower third y ticker
- Posición izquierda/centro/derecha
- Colores personalizables
- Vista previa interna
